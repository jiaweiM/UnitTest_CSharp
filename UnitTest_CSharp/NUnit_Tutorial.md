NUnit 是 .NET 平台通用的单元测试框架。

# 快速入门
在Visual Studio 平台使用NUnit，建议先安装 NUnit 3 Test Adapter 和 NUnit VS Template，
这样方便快速的创建测试项目和类。

## 安装 NUnit
依次选择 Tools|NuGet Package Manager|Manage NuGet Packages for solution...
打开 Browser 选项，搜索 NUnit，添加即可。
